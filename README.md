# Topfunky Tap

## How do I install these formulae?

`brew install topfunky/tap/<formula>`

Or `brew tap topfunky/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "topfunky/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
